The Online Serving Pipeline

What happens every time a user asks a question — 7 stages in under 5 seconds

The latency budget

The entire pipeline must complete in under 5 seconds end-to-end. Every stage adds latency, so every stage must earn its place. Streaming the LLM response delivers first tokens at ~1.5s, making the system feel fast even if total latency is 4.5s.

The seven stages

  1. Query understanding (~100ms): Rewrite the user's colloquial question into search-friendly form. HyDE generates a hypothetical answer and embeds that instead of the raw question — it sits closer to real document passages in embedding space.
  2. Intent classification (~10ms): Is this query in scope? A lightweight classifier catches off-topic questions before expensive retrieval runs. "I don't cover that" costs 10ms; a failed retrieval costs 800ms.
  3. Access control filter (~5ms): Scope retrieval to the user's permission set. Enforced server-side before any index query. Must be chunk-level, not document-level — documents can have restricted appendices with public preambles.
  4. Hybrid retrieval (~50ms): Run dense ANN + sparse BM25 in parallel. Fuse results with Reciprocal Rank Fusion (RRF), which combines by rank position, not raw scores. Output: ~80 fused candidates.
  5. Cross-encoder re-ranking (~300ms): Score each candidate jointly with the query using token-level attention. MiniLM scores 80 candidates in ~300ms. Output: top 5-10.
  6. Context assembly (~50ms): Expand top chunks by pulling neighboring chunks for context. Compress by scoring each sentence for relevance and discarding noise. Less context = fewer hallucinations.
  7. LLM generation (~2-4s): Stream the answer with mandatory citations. System prompt enforces: "answer only from provided context; if the answer isn't there, say so."

Pipeline orchestration

Stages 1-3 run sequentially (each depends on the previous). Stage 4 runs two retrievers in parallel. Stage 5-6 run sequentially on the fused results. Stage 7 streams to the user while faithfulness scoring runs in parallel on completed sentences.

Think of retrieval as casting a wide net (80 candidates) and re-ranking as a careful hand-sort of the catch. The LLM is the chef who turns the best fish into a meal — but only what's on the plate in front of it.
