LLM Hallucination

Why models confidently invent facts — and what you can do about it

What is hallucination?

An LLM "hallucinates" when it generates text that sounds plausible but is factually incorrect, fabricated, or unsupported by any training data. The model does not know it is wrong — it has no internal fact-checker. It simply predicts the most probable next token given everything before it.

A concrete example

Suppose you ask: "What is the return policy for Acme Corp?"

An LLM without access to Acme's actual documents might respond: "Acme Corp offers a 30-day return policy with full refund on all items." This sounds confident and professional — but it may be entirely made up. The real policy might be 14 days, or only on specific product lines, or require the original receipt.

Why does it happen?

  • Statistical prediction, not knowledge retrieval: LLMs predict text based on patterns seen during training. They optimize for fluency and coherence, not factual accuracy.
  • Training on noisy data: The internet contains contradictory and outdated information. The model absorbs all of it without a mechanism to verify truth.
  • No epistemic uncertainty: The model has no way to say "I'm 30% sure about this." It always generates with the same apparent confidence.
  • Instruction-following pressure: When asked to answer, the model wants to comply. Saying "I don't know" is rarely the most probable completion.

How RAG mitigates hallucination

RAG addresses this by grounding the model's generation in retrieved evidence. Instead of asking "what do you know about X?", a RAG system says "here are the relevant documents — answer based only on these."

  • The LLM generates from provided context, not from memory
  • Citations can be verified by the user
  • If no relevant document is found, the system can say "I don't have information on that" instead of guessing

Key takeaway: hallucination is not a bug in the model — it is a fundamental property of how language models work. RAG does not eliminate hallucination entirely, but it shifts the problem from "inventing facts from nothing" to "misinterpreting provided evidence," which is far more controllable.
